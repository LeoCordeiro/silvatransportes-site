# WFM Transportes

Site institucional da **WFM Transportes**.

## Tecnologia
- Vue 3 + Vue Router
- Vuetify 3 + Tailwind CSS
- Cores: #F97316 (laranja) + #1C1C1E (preto)

## Instalação
```bash
npm install
npm run serve
```

## Build
```bash
npm run build
```
